# Campus Recruiting Portal

## 📌 Overview
The Campus Recruiting Portal is designed to connect **students, companies, and administrators** in a streamlined recruitment process.  
It provides a platform where:
- Students can apply for jobs and internships.
- Companies can post opportunities and review applicants.
- Administrators can manage approvals and oversee the recruitment workflow.

---

## 🎯 Features
- **Student Dashboard**: Apply for jobs, track application status.
- **Company Dashboard**: Post openings, shortlist candidates.
- **Admin Panel**: Approve/reject postings, monitor recruitment activity.
- **Authentication**: Secure login for all roles.
- **Responsive UI**: Built with modern front‑end frameworks for smooth experience.

---

## 🛠️ Tech Stack
- **Frontend**: React (with reusable components)
- **Backend**: Node.js 
- **Database**: MongoDB
- **Other Tools**: REST APIs, JWT authentication

---

Navigate into the project folder:
cd campus-recruiting-portal

Install dependencies:
npm install
Start the development server:
npm start
